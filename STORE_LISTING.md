### ## English (en)

PagePalette - Custom CSS Injector

**Short Description (max 132 chars):**
Customize your favorite websites by automatically injecting specific CSS styles based on URL rules. 100% private and offline.

**Description:**
PagePalette is a lightweight browser extension that lets you apply custom CSS styles to any website. Whether you want to hide annoying banners, create a makeshift dark mode for a specific site, or just tweak the font size, PagePalette gives you full control.

**Features:**
* **Regex URL Matching:** Apply styles to a single page or entire domains using powerful Regular Expressions (e.g., `.*github\.com.*`).
* **Easy Management:** An intuitive, clean options page to add, edit, toggle, and delete your rules.
* **Instant Application:** Changes apply seamlessly as the page loads.
* **Privacy First:** 100% offline. No data is sent to external servers or developers. Your rules synchronize securely via your browser account.
* **Lightweight:** Minimal permissions and optimized background processing.

### ## Deutsch (de)

PagePalette - CSS Styling Extension

**Kurzbeschreibung (max 132 Zeichen):**
Passe deine Lieblings-Websites an, indem du automatisch CSS-Stile basierend auf URL-Regeln einfügst. 100 % privat und offline.

**Beschreibung:**
PagePalette ist eine schlanke Browser-Erweiterung, mit der du benutzerdefinierte CSS-Stile auf jede beliebige Website anwenden kannst. Egal, ob du nervige Banner ausblenden, einen eigenen Dark Mode für eine bestimmte Seite basteln oder nur die Schriftgröße anpassen willst – PagePalette gibt dir die volle Kontrolle.

**Funktionen:**
* **URL-Muster mit Regex:** Wende Stile nur auf eine einzige Seite oder gleich auf ganze Domains an, indem du Reguläre Ausdrücke nutzt (z. B. `.*wikipedia\.org.*`).
* **Einfache Verwaltung:** Eine übersichtliche Options-Seite, um deine Regeln hinzuzufügen, zu bearbeiten, an-/auszuschalten und zu löschen.
* **Sofort wirksam:** Änderungen werden nahtlos angewendet, sobald die Seite lädt.
* **Privatsphäre zuerst:** 100 % offline. Es werden keine Daten an externe Server oder den Entwickler gesendet. Deine Regeln synchronisieren sich sicher über dein Browser-Konto.
* **Ressourcenschonend:** Minimale Berechtigungen und optimierte Ausführung im Hintergrund.
